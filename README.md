# Projeto Viva bem
O Viva Bem é uma aplicação web voltada à conscientização e prevenção em saúde, promovendo o autocuidado, exames preventivos e informações importantes sobre bem-estar físico e mental.

O site apresenta conteúdos de forma simples, visual e responsiva, destacando cuidados femininos, masculinos e alertas gerais para todas as idades.
